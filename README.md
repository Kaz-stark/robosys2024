## 概要
-  このプロジェクトは、標準入力で得た数字の分散と標準偏差を計算して出力します。    　


## 使い方
以下の手順で実行する。

```
リポジトリをクローン
$git clone https://github.com/Kaz-stark/robosys2024.git

ディレクトリに移動
$cd robosys2024

実行権限を付与する
$chmod +x robosys2024
```
